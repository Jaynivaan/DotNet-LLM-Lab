##gs

Token	-	Smallest unit processed by an LLM
Tokenizer	-	Converts text into tokens
Transformer		-	Neural network architecture used by modern LLMS.
Parameter	-	A Learned weight inside the model
CheckPoint	-	Saved Model State.
FineTuning	-	Continuing Training on a Pretrained model.
Inference	-	Using a trained model to generate Predictions.
HugingFace hub	-	Repository of AI models and datasets.
